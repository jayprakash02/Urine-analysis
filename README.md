# Urine Strip Analysis

## Open CV + Flask

### Setup

* If you use Conda

  ```python
  conda env create --file=environment.yml
  python Machine_learning.py
  ```

    Now you do post request at``/uploads`` with ``img`` as file attribute.

* If you don't use conda
  * See req.txt
  * Download requirement listed there
  * ```
    python Machine_learning.py
    ```

    Now you do post request at``/uploads`` with ``img`` as file attribute.
